Operating Systems Project 3 Part 2


This project folder contains two C++ files and one C file, each with
its own main method. Each is named after the part of the project and
an executable for each is included with the same name.

farmers.c contains the code compiled to the farmers executable, done for
the first part of the project.

bankers.c contains the code compiled to the bankers executable, done for
the second part of the project on the banker's algorithm.

paging.c contains the code compiled to the paging executable, done for
the third part of the project on paging.
